# iOS Project

-----------------------

## Name: 
VideoSticker

## Description:
This project show how to add many stickes to a video. You can move, scale, rotate every sticker.


-----------------------

## 简介:
这个工程演示了在iOS平台上如何控制在视频上添加多个贴纸，贴纸可以移动、缩放、旋转，最后生成新的视频放到相册中。
